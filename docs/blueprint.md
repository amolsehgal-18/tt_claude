# **App Name**: Touchline Tantrum

## Core Features:

- Manager Dashboard & Tension Triangle Tracking: Display the 'Tension Triangle' stats (Board Support, Fan Support, Dressing Room, Aggression), represented as decimal values between 0.01 and 1.0 (shown as percentages) on a central dashboard with clear visualizations. The manager's dynamic mood (happy, neutral, stressed, angry, sacked) will be a visual summary, calculated based on the Tension Triangle stats, season objectives, and recent performance.
- AI Scenario Engine & Decision Making: Present interactive, dynamically generated scenarios from an AI tool ('gemini-1.5-flash' via 'AIService') to the user. The AI leverages current game context (User Team, Board Trust, Fan Support, Dressing Room, Current Aggression, Current League Position, Saga Objective) to create relevant dilemmas. Key generation rules include: (1) Stat-Awareness: Prioritize 'Crisis' or 'Financial Audit' cards if Board Trust < 30%; 'Player Revolt' or 'Training Ground Fight' if Dressing Room < 30%; and 'Ego' or 'Media Distraction' cards if the user is meeting their Objective. (2) No Repetition: Scenarios will not be repeated. (3) Output Format: The AI returns a JSON object like: {"scenario": "string", "leftOption": "string", "rightOption": "string", "impactLeft": {"board": int, "fans": int, "squad": int, "aggression": double}, "impactRight": {"board": int, "fans": int, "squad": int, "aggression": double}, "imageCategory": "string", "isBreaking": bool}. (4) Math Constraints: Impacts on Board, Fans, and Squad range from -20 to +15. Aggression shifts range from -0.1 to +0.1. (5) Themes: Scenarios will mix Tactical, Satirical, Foundation, and Breaking themes. A 15-second countdown timer will force a decision, automatically applying the 'Left Option' if it expires. The AI is informed by a comprehensive dataset of scenario cards, similar to the provided CSV data, to ensure varied and realistic dilemmas.
- Scenario Impact & State Update: Process the chosen scenario option (swiping Left for 'leftOption', Right for 'rightOption'), applying its specific impacts (from the 'impactLeft' or 'impactRight' objects) to the four Tension Triangle stats (Board Support, Fan Support, Dressing Room, Aggression). The impacts for Board Support, Fan Support, and Dressing Room will be integers (e.g., -20 to +15), while Aggression changes will be doubles (e.g., -0.1 to +0.1). The game's core state and manager mood will be updated accordingly. Note: Swiping right applies the 'rightOption' impacts directly.
- Match Simulation & Outcome Display: Every 3 scenario cards, trigger a 'Nitro Reel' simulation. Calculate Win Probability using the formula: 0.30 + (Aggression * 0.20) + (DressingRoom * 0.20). Display an animated 'Match Radar' using CustomPaint showing moving player dots during simulation, leading to simplified outcomes. Match odds for Win/Draw/Loss will be dynamically displayed based on the aggression stat: 'Win | Draw | Loss' (e.g., "${(2.20 - (aggression * 0.8)).toStringAsFixed(2)} | 3.40 | ${(2.40 + (aggression * 1.2)).toStringAsFixed(2)}").
- Local & Cloud Persistence: Save game progress locally using SharedPreferences. Synchronize lifetime wins and sackings to a Cloud Firestore-backed 'Trophy Cabinet' for long-term tracking.
- The Resume Gate: On the Landing Screen, check for existing local (SharedPreferences) or Firestore-synced save data. Display a 'RESUME' button only if a save exists, allowing users to continue their previous game.
- VAR Wildcard: On loss screens, a 'Call VAR' button will be available. If pressed, it triggers a placeholder for a Rewarded Ad, offering a chance to overturn the match result.
- Sack Summary & Social Sharing: Generate a visual 'Sack Summary' widget upon game conclusion or a sacking event. This widget will capture the current screen as an image for sharing on social media, detailing the manager's fate.
- Connectivity Safety Net: If connectivity is lost, the 'Call VAR' button will be hidden, and gameplay will be limited to 2 Game Weeks (Weeks 36 & 37) to ensure a controlled experience during offline periods.

## Style Guidelines:

- Primary UI elements (buttons, containers) will feature a 'Premium White' with 0.1 opacity, allowing the deep stadium background to subtly show through.
- The overall app background will be a 'deep stadium background', creating an immersive and atmospheric visual foundation.
- Accent colors for interactive elements, urgent feedback, and highlights will use a vibrant, striking orange (#FFAD1F) to provide clear contrast and urgency.
- Headlines will utilize 'Space Grotesk' (sans-serif) for its modern, slightly technical, and impactful feel, aligning with the game's strategic nature. Body text will use 'Inter' (sans-serif) for its neutrality and excellent readability.
- Clean, sharp, and angular icons that complement the 'Slanted UI' aesthetic, ensuring readability and consistency with the modern sports management theme.
- Emphasize a 'Slanted UI' aesthetic using CustomClippers for all buttons and containers, creating dynamic and visually engaging asymmetric layouts for core sections like dashboards and scenario prompts.
- Subtle and fluid animations for transitions, scenario presentations, visual feedback (e.g., 15-second countdown timer), match progress indicators (e.g., 'Match Radar'), and tension stat changes, to enhance the user experience.
